# 企画書

## Diff Sync Code
[https://diff.yumekiti.net/](https://diff.yumekiti.net/)

## 概要
Diff Sync Codeは、複数人でコーディングの課題に取り組む際に、
リアルタイムで差分を取りながらコードを共有・比較するためのWebアプリケーションです。
A君のコードでは動くけれどB君のコードでは動かないといった問題を解決し、効率的なチームのコーディング作業を実現します。

## 問題定義
学校の課題のコードを書いていると、チームメンバーのコードが動かない場合、
現在はdifffというサイトを使って差分をとり構文ミスを見つけています。
しかし、このサイトでは1つのパソコンに2人のコードを入れる必要があり、手間がかかってしまいます。
そこで、リアルタイムで複数人のパソコン間で差分を取れるアプリケーションを制作することにしました。

## 解決策
Diff Sync Codeでは、2人以上のエンジニアが同時にコーディングを行いながら、
リアルタイムで差分を取ることができます。各メンバーが独自のエディタでコードを入力し、
差分が共有されることで、構文ミスやロジックのミスを即座に発見し、解決することができます。
これにより、チームのコーディング作業の効率と品質を向上させることができます。
